# Heartbeat Checklist

## Task Progress
- [ ] Review current task status — am I blocked? Do I need to escalate?
- [ ] Check for new comments on the active issue
- [ ] Update task state if progress was made since last heartbeat

## Memory Hygiene
- [ ] Any new conventions or patterns discovered? Add to MEMORY.md
- [ ] Any corrections received? Record them

## Communication
- [ ] Post progress update to issue if meaningful work was done
- [ ] Flag blockers early — don't wait for the next heartbeat
