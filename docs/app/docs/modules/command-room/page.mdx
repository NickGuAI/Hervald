# Command Room

The Command Room is a cron-based task scheduler for recurring agent operations. It manages scheduled tasks, tracks run history, and supports manual triggers.

## Features

### Task Scheduling

Tasks are defined with standard **cron expressions**:

```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ minute (0-59)
в”Ӯ в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ hour (0-23)
в”Ӯ в”Ӯ в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ day of month (1-31)
в”Ӯ в”Ӯ в”Ӯ в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ month (1-12)
в”Ӯ в”Ӯ в”Ӯ в”Ӯ в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ day of week (0-7, Sun=0 or 7)
в”Ӯ в”Ӯ в”Ӯ в”Ӯ в”Ӯ
* * * * *
```

Common patterns:

| Expression | Schedule |
|-----------|----------|
| `0 9 * * *` | Every day at 9:00 AM |
| `*/15 * * * *` | Every 15 minutes |
| `0 0 * * 1` | Every Monday at midnight |
| `0 */6 * * *` | Every 6 hours |

### Task Configuration

Each task specifies:

- **Name** вҖ” human-readable identifier
- **Schedule** вҖ” cron expression
- **Instruction** вҖ” the prompt or command to execute
- **Agent type** вҖ” `claude` or `codex`
- **Session type** вҖ” `stream` or `pty`
- **Permission mode** вҖ” tool approval behavior
- **Working directory** вҖ” execution context
- **Machine** вҖ” target machine for multi-host setups
- **Enabled/Disabled** вҖ” toggle without deleting

### Run History

Each task maintains a history of recent executions with:
- Execution timestamp
- Duration
- Exit status (success/failed)
- Output excerpt

### Manual Trigger

Any task can be triggered immediately regardless of its schedule. Use the dashboard button or the CLI:

```bash
hambros cron trigger --instruction "Run the daily report now"
```

## Creating Tasks

### Via Dashboard

1. Open **Command Room** in the navigation
2. Click **New Task**
3. Fill in the schedule, instruction, and configuration
4. Save

### Via CLI

```bash
hambros cron add \
  --commander <commander-id> \
  --schedule "0 9 * * *" \
  --instruction "Review open PRs and summarize status" \
  --agent claude \
  --name "daily-pr-review"
```

### Via API

```bash
curl -X POST http://localhost:20001/api/command-room/tasks \
  -H "Content-Type: application/json" \
  -H "X-Hammurabi-Api-Key: YOUR_KEY" \
  -d '{
    "name": "daily-pr-review",
    "schedule": "0 9 * * *",
    "instruction": "Review open PRs and summarize status",
    "agentType": "claude"
  }'
```
