# Getting Started

Get HamBros running locally in under five minutes.

## Prerequisites

- **Node.js** 20 or later
- **pnpm** 9 or later
- A terminal emulator (iTerm2, Terminal.app, Windows Terminal)

## Quick Start

```bash
# Install or upgrade HamBros
curl -fsSL https://raw.githubusercontent.com/NickGuAI/HamBros/main/install.sh | bash

# Configure your local instance
hambros init

# Start HamBros
hambros start
```

This installs HamBros into **`~/.hambros`** by default, adds the `hambros` CLI to your `PATH`, and serves the dashboard at **http://localhost:20001**.

If you want to work from a local clone instead of the installer-managed setup, use the [Installation](/docs/getting-started/installation) guide's source workflow.

## Default Authentication

On first boot, HamBros creates a **master API key** stored at `app/data/api-keys/master-key.json`. The default password is:

```
HAMBROS!
```

> **Security note**: Delete the master key file and create a new API key before deploying to production.
